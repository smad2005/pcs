pcs
===

Client for http://tknkk.tk/ (former pcs.org.ua and img.if.ua)

For install:
1.Click on PcsDownloader.exe
2.Profit

For configuration:
-Open GuiConfig.exe

How use:
-Right click on image and select in contex menu "Залить на pcs"
